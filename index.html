Prototype & jQuery Excercise

In this excercise you need to make jQuery working on the same page with prototype.

Things to do:

  1. Make script from main.js working (it needs to add class to the body element).
  2. You can wrap the code, add something before and something after but you can't change the line: 
    $('body').addClass('jquery-is-working')
  3. Second part of the task is to make mark.js plugin working, HTML is already added, mark.js is also added (plugins.js), you must initialise the script in main.js
  4. Mark.js script should work like here: https://jsfiddle.net/julmot/vpav6tL1/
  5. Of course you can't remove prototype.js, the clicking script below needs to work.
